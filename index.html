¿Qué necesito para hacer una pagina web?

Para hacer una pagina web se necesitan determinadas herramientas para su creación

Obviamente como el mundo está tan evolucionado, podemos encontrar señal de internet en gran parte del mundo, pero hoy en día se necesita de un servicio para tenerlo. Lo mismo pasa con los dispotivos electronicos.

REQUISITOS

Lo que vamos a necesitar para hacer una pagina web es:

  1. computora (pc o laptop)
  2. conexion a internet

COMENTARIO

Obviamente no tener computadora no implica que una persona no pueda aprender desarrollo web. El conocimiento es aparte del saber ya que uno puede conocer la teoria pero saber la practica lo mejora a uno

EDITOR DE TEXTO

Para poder crear la pagina web vamos a necesitar un programa en especial para poder escribir codigo.

A continuacion voy a poner los editores de texto que yo recomiendo para usar:

  • Sublime Text
  • Atom
  • Visual Studio Code
  • Emacs
  • Vim

Si bien estos editores son gratuitos, cada uno tiene su propieda dificultad o facilidad de uso, que da su criterio elegir cual usar

Para este turorial vamos a ver la instalacion de Sublime Text
